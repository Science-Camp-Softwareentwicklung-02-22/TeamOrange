# TeamOrange

A little top down shooter written in a week.

## Commands

### Installation

```
git clone https://github.com/Science-Camp-Softwareentwicklung-02-22/TeamOrange
cd TeamOrange
npm install
```

### Development Build

```
npm run build_debug
npm run dev
```

### Release Build

```
npm run build_release
npm run serve
```
