# TeamOrange

## Commands

### Installation

```
git clone https://github.com/Science-Camp-Softwareentwicklung-02-22/TeamOrange
cd TeamOrange
npm install
```

### Development Build

```
npm run build_debug
```

### Release Build

```
npm run build_release
```
